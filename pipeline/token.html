# Token

# PipelineToken

Token class used among all the stores in the pipeline. Typically generated from create_token() from PyotCore objects.

# __init__(model: str, server: str, method: str, params: Dict[str, Any], queries: Dict[str, Any])

  • model param -> str: Name of the pipeline.
  • server param -> str: Name of the server (region/platform/locale).
  • method param -> str: Name of the method endpoint.
  • params param -> Dict[str, Any]: Dict containing the params.
  • queries param -> Dict[str, Any]: Dict containing the queries params.

# __hash__(self)

Returns the hash of the token.

# stringify property

Returns the string version of the token.

# RequestToken

Token class used to handle request based on configured error handler.

# __init__(tries: int = 0, _raise_at: int = 1, _exception: Any = None, _code: int = 0)

  • tries param -> int: Number of tries attempted.
  • _raise_at param -> int: Number of try to raise exception.
  • _exception param -> Any: Exception to raise.
  • _code param -> int: Status code of the request.

# stream(code: int, how: Tuple, origin: str)

Streams the new state to the token.

  • code param -> int: Status code of the request.
  • how param -> Tuple: Tuple containing the handler configs.
  • origin param -> str: Attempted request's pipeline token string.

# run_or_raise() function awaitable

Decide if the the request should be made or raise exception.
